This repository contains:

    "myocyte_creation_diameterOrienHeterogeneity" matlab code to generate the virtual cardiac tissue model with heterogeneity diameters and orientation of cardiac myocytes.
    "Model_exemple" a example of virtual model.
    "C code" C++ code to process Monte Carlo simulation within virtual cardiac tissue model.
    "Observation" matlab code to observe the results from Monte Carlo simulation and generate the PGSE virtual dMRI signal and "Observation_noise" to generate the virtual signal with noise.
    "DTI" matlab function code is used in "Observation" to reconstructe the diffusion tensor and corresponding parameters.
