This repository contains:

    "C code" C++ code to process Monte Carlo simulation.
    "Model" a example of virtual model, it should be read by "C code".
    "Observation" matlab code to observation the results from Monte Carlo simulation and "Observation_noise" to generate the virtual signal with noise.
    "DTI" matlab function code is used in "Observation" to reconstructe the diffusion tensor and corresponding parameters.
